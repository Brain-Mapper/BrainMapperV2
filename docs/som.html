6.2. Self-organizing maps in BrainMapper

To use the self-organizing maps in BrainMapper, you have to import a CSV or an Excel file in the main windows and then to select the button "SOM".

6.2.1. Parameters

In the SOM windows, you can choose many paramters before training the maps.

You can also select data columns for the self-organizing maps. In the following example, we select the columns X and Y.

NOTE
You can train the maps on the coordinates (X, Y, Z) or on others columns. In the previous example, you could have selected the type or the subtype.

6.2.2. Training

Once you have chosen the parameters, you can launch the training. To do, you have to select the button "Train".

When the training is finished, a new windows will appear to announce the end of the training.

NOTE
The training can be quite long.
